```python
from pydantic import BaseModel, Field
from typing import List, Literal

class WyckoffEvent(BaseModel):
    candle_index: int = Field(ge=0)
    event_type: Literal["spring", "upthrust", "sos", "sow", "lps", "lpsy"]
    price: float = Field(gt=0)
    volume_ratio: float = Field(gt=0)
    description: str = Field(min_length=1)

class WyckoffAnalysis(BaseModel):
    phase: Literal[
        "accumulation_a", "accumulation_b", "accumulation_c", "accumulation_d", "accumulation_e",
        "distribution_a", "distribution_b", "distribution_c", "distribution_d", "distribution_e",
        "unknown"
    ]
    phase_confidence: int = Field(ge=0, le=100)
    events: List[WyckoffEvent] = Field(default_factory=list)
    volume_confirms: bool
    analysis_notes: str = Field(default="")
```

  <action>
Add to wyckoff.py:

**Helper function: _classify_phase(df, events, sr_levels)**
Determine current Wyckoff phase from event sequence and price position:

- **Phase A (Stopping Action):** Recent high-volume reversal candle near support/resistance. Look for climax volume > 2x baseline in last 20 candles.
- **Phase B (Building Cause):** Range-bound (price within support/resistance), volume declining vs prior 50 candles. No strong directional events.
- **Phase C (Test):** Spring or Upthrust detected. If spring -> accumulation_c. If upthrust -> distribution_c.
- **Phase D (SOS/SOW):** SOS or SOW detected. If SOS -> accumulation_d. If SOW -> distribution_d.
- **Phase E (Trending):** Price outside range after SOS/SOW. Markup (price > resistance) or Markdown (price < support).

Return tuple: (phase: str, confidence: int)

**Helper function: _check_volume_confirmation(df, phase, volume_baseline)**
- For accumulation phases: volume should decline during B, expand during C/D events
- For distribution phases: similar inverse pattern
- Return bool

**Helper function: _calculate_confidence(df, events, phase)**
Base confidence 50, adjust:
- +20 if len(df) >= 100
- -20 if len(df) < 50
- +15 if 2-8 events detected
- -30 if 0 events
- -20 if > 15 events (over-detection)
- +15 if phase != "unknown"
- Clip to 0-100

**Main function: detect_wyckoff(df: pd.DataFrame) -> Optional[WyckoffAnalysis]**
Entry point per CONTEXT.md:
1. Return None if df is None or len(df) < 100
2. Get support/resistance via detect_support_resistance(df, num_levels=3)
3. Calculate volume_baseline via _calculate_volume_baseline(df, period=20)
4. Detect all events: spring, upthrust, sos, sow
5. Classify phase via _classify_phase
6. Check volume confirmation via _check_volume_confirmation
7. Build analysis_notes with detected event count
8. Return WyckoffAnalysis model

Wrap main logic in try/except, return None on any exception.
