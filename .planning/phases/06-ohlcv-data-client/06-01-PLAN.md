3. Create `src/backend/data/ohlcv_client.py` with:
   - OHLCVClient class that initializes ccxt.binance() exchange (public API, no API key needed)
   - SUPPORTED_SYMBOLS constant: ["BTC/USDT", "ETH/USDT", "SOL/USDT"]
   - SUPPORTED_TIMEFRAMES constant: ["1w", "1d", "4h"]
   - `fetch_ohlcv(symbol: str, timeframe: str, limit: int = 100) -> list[dict]` method:
     - Validates symbol in SUPPORTED_SYMBOLS
     - Validates timeframe in SUPPORTED_TIMEFRAMES
     - Calls `exchange.fetch_ohlcv(symbol, timeframe, limit=limit)`
     - Converts CCXT array format to dict format: {timestamp, open, high, low, close, volume}
   - Retry decorator or wrapper with exponential backoff:
     - Catch ccxt.RateLimitExceeded and ccxt.RequestTimeout
     - Base delay: 1 second
     - Max retries: 3
     - Exponential factor: 2 (delays: 1s, 2s, 4s)
     - Optional jitter up to 500ms
   - `fetch_all_timeframes(symbol: str) -> dict` convenience method:
     - Returns {"1w": [...], "1d": [...], "4h": [...]} for given symbol
   - Singleton pattern: `get_ohlcv_client() -> OHLCVClient`
   - Include type hints throughout
   - Add docstrings explaining each method

Important: Use ccxt.binance() without API key - public OHLCV data does not require authentication.
