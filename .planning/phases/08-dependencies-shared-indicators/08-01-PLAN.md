**src/backend/analysis/indicators.py:**
Implement all 7 indicator functions as pure functions. Each function:
- Takes pd.DataFrame with columns: timestamp, open, high, low, close, volume (lowercase)
- Returns scalar value or dict (no side effects)
- Returns None if insufficient data (with no exception)
- Uses pandas-ta for calculations

Parameters (from CONTEXT.md locked decisions):
- RSI: period 14
- MACD: fast 12, slow 26, signal 9
- Bollinger Bands: period 20, std dev 2
- ADX: period 14
- OBV: standard calculation
- VWAP: standard calculation (requires DatetimeIndex - handle by converting timestamp column)
- ATR: period 14

Implementation notes from research:
- pandas-ta expects lowercase column names (our OHLCVClient already uses lowercase)
- For VWAP: convert timestamp to DatetimeIndex before calculation
- pandas-ta returns Series for single-value indicators (RSI, ADX, ATR, OBV, VWAP)
- pandas-ta returns DataFrame for multi-value indicators (MACD, Bollinger Bands)
- Always use `.dropna().iloc[-1]` to get most recent non-NaN value
- Handle edge case where result is empty after dropna()

Minimum data requirements (document in docstrings):
- RSI: period + 1 candles (15 for period=14)
- MACD: slow + signal candles (35 for defaults)
- Bollinger Bands: period candles (20 for period=20)
- ADX: period + 1 candles (15 for period=14)
- OBV: 1 candle minimum
- VWAP: 1 candle minimum
- ATR: period + 1 candles (15 for period=14)

import pandas as pd
import numpy as np
from src.backend.analysis import (
    calculate_rsi, calculate_macd, calculate_bollinger_bands,
    calculate_adx, calculate_obv, calculate_vwap, calculate_atr
)

# Generate 100 candles of synthetic OHLCV data
np.random.seed(42)
n = 100
base_price = 50000.0
returns = np.random.normal(0.001, 0.02, n)
prices = base_price * np.exp(np.cumsum(returns))

df = pd.DataFrame({
    'timestamp': pd.date_range(start='2024-01-01', periods=n, freq='1D'),
    'open': prices * (1 + np.random.uniform(-0.005, 0.005, n)),
    'high': prices * (1 + np.random.uniform(0.005, 0.015, n)),
    'low': prices * (1 + np.random.uniform(-0.015, -0.005, n)),
    'close': prices,
    'volume': np.random.uniform(1000, 5000, n)
})

# Test each indicator
print(f"RSI: {calculate_rsi(df)}")  # Should be 0-100
print(f"MACD: {calculate_macd(df)}")  # Should be dict with macd, signal, histogram
print(f"Bollinger: {calculate_bollinger_bands(df)}")  # Should be dict with upper, middle, lower
print(f"ADX: {calculate_adx(df)}")  # Should be >= 0
print(f"OBV: {calculate_obv(df)}")  # Should be numeric
print(f"VWAP: {calculate_vwap(df)}")  # Should be numeric
print(f"ATR: {calculate_atr(df)}")  # Should be > 0

# Test insufficient data handling
small_df = df.head(5)
print(f"RSI (5 candles): {calculate_rsi(small_df)}")  # Should be None
print(f"MACD (5 candles): {calculate_macd(small_df)}")  # Should be None
