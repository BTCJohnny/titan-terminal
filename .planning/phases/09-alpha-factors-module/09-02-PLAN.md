   - test_returns_dict_with_correct_keys
   - test_is_anomaly_true_when_above_threshold
   - test_is_anomaly_false_when_below_threshold

   - test_zero_avg_volume_returns_none
   - test_custom_threshold

3. **TestMADeviation**
   - test_returns_dict_with_correct_keys

   - test_price_above_emas_positive_deviation
   - test_price_below_emas_negative_deviation

4. **TestVolatilityScore**
   - test_returns_dict_with_correct_keys
   - test_volatility_score_in_range (assert 0 <= score <= 100)
   - test_insufficient_data_returns_none (< 15 candles)
