```python
def get_signals_connection() -> sqlite3.Connection:
    """Get connection to signals.db with row factory."""
    db_path = Path(settings.SIGNALS_DB_PATH)
    conn = sqlite3.connect(str(db_path))
    conn.row_factory = sqlite3.Row
    return conn
```

  <action>
Add the core TelegramAgent.analyze(symbol: str) method that returns TelegramSignal:

1. **Call _query_signals** to get recent signals

2. **Handle empty results** - return TelegramSignal with:
   - signals_found=0, active_signals=0
   - relevant_signals=[]
   - overall_sentiment="neutral"
   - confluence_count=0, confidence=0, avg_confidence=0.0
   - best_signal=None
   - reasoning="No recent Telegram signals found for {symbol}"

3. **Build TelegramChannelSignal list** from database rows:
   - Map `direction` to `action` ("long"/"short"/"buy"/"sell")
   - Extract entry_price from first non-null of entry_1, entry_2, entry_3
   - Extract stop_loss directly
   - Build take_profits list from target_1 through target_5 (filter out nulls)
   - Map confidence_score to signal_quality (0-100)
   - Set freshness to "fresh" if created_at within 12 hours, else "stale"
   - Set channel to provider field
   - Set raw_text to setup_description if available

4. **Calculate confluence** (TELE-04):
   - Count signals by direction (long/buy vs short/sell)
   - bullish = count of long + buy
   - bearish = count of short + sell
   - confluence_count = max(bullish, bearish)

5. **Determine overall_sentiment**:
   - If bullish > bearish: "bullish"
   - If bearish > bullish: "bearish"
   - If bullish == bearish and both > 0: "mixed"
   - Else: "neutral"

6. **Calculate confidence**:
   - avg_confidence = mean of all signal confidence_scores
   - confidence = int(avg_confidence * (confluence_count / total_signals)) if signals exist

7. **Find best_signal** (TELE-05):
   - Select TelegramChannelSignal with highest signal_quality

8. **Build reasoning string**:
   - Format: "{N} signals found. {confluence_count} signals agree on {direction}. Best signal from {provider} with {confidence}% confidence."

9. **Return TelegramSignal** with all fields populated.

Import TelegramSignal and TelegramChannelSignal from models:
