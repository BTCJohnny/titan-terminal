   - Assert `suggested_action` in `["Long Spot", "Long Hyperliquid", "Short Hyperliquid", "Avoid"]`
   - Assert `accumulation_score` and `distribution_score` are int, 0-100
   - Assert `timestamp` is a non-empty string
   - If `direction` is not None, assert it is in `["BULLISH", "BEARISH", "NO SIGNAL"]`
   - If `suggested_action != "Avoid"`: assert `reasoning` is a non-empty string, `entry_zone` is a dict with keys `low`, `high`, `ideal`, `stop_loss` is a float, `tp1` is a float, `tp2` is a float, `risk_reward` is a float >= 0
   - If `three_laws_check` is not None: assert it has keys `law_1_risk`, `law_2_rr`, `law_3_positions`, `overall`

   - GET `/api/health-test` → assert 200

   - For each signal in `signals`, call `_assert_valid_signal(signal)`
