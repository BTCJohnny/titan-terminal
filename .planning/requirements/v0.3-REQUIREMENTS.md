# v0.3 TA Ensemble Requirements

**Milestone:** v0.3 TA Ensemble
**Created:** 2026-02-27
**Status:** Approved

## Scope

Implement fully functional TA subagents (Weekly, Daily, 4H) with Wyckoff detection, technical indicators, and alpha factors. TAMentor synthesizes all three with conflict resolution.

**In Scope:**
- WeeklySubagent, DailySubagent, FourHourSubagent
- TAMentor with Anthropic SDK
- Shared analysis modules (indicators, wyckoff, alpha_factors)
- Extended TASignal model
- Unit tests with mocked OHLCV

**Out of Scope:**
- Nansen agent
- Telegram agent
- Risk agent
- Integration tests
- Production deployment

## Decisions Made

| Decision | Rationale |
|----------|-----------|
| pandas-ta over TA-Lib | Pure Python, no C dependencies, easier installation |
| Shared modules pattern | Avoid code duplication across 3 subagents |
| Anthropic SDK direct for TAMentor | Simpler than full agent framework for synthesis task |
| Extend TASignal vs new model | Backward compatible, all subagents use same model |
| Optional wyckoff/alpha_factors fields | Graceful degradation if analysis fails |
| 2 years OHLCV history | Sufficient for Wyckoff phase detection |
